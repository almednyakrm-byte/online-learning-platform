# منصة تعليمية إلكترونية للتعلم عن بعد (دورات، اختبارات، تفاعل مع المعلم)

## ملخص المشروع

منصة تعليمية إلكترونية للتعلم عن بعد، توفر دورات تعليمية، اختبارات، وتفاعل مع المعلمين، لتسهيل عملية التعلم عن بعد.

### الأهداف الرئيسية للمشروع

- توفير منصة تعليمية إلكترونية متكاملة.
- تسهيل عملية التعلم عن بعد.
- تقديم دورات تعليمية متقدمة.
- تقديم اختبارات ومراجعات للطلاب.
- توفير تفاعل مع المعلمين.

## هيكلة المشروع

### مجالات المشروع

- **backend**: تطبيقات الخادم.
- **frontend**: تطبيقات المستخدم.
- **database**: قواعد البيانات.
- **docker**: تطبيق الحاويات.

## تثبيت وتشغيل المشروع

### تثبيت الحاويات

bash
docker-compose up -d


### تشغيل المشروع

bash
docker-compose exec backend python app.py


## واجهة المستخدم

### دورات التعلم

- **دورة 1**: دورة تعلم أساسيات البرمجة.
- **دورة 2**: دورة تعلم تصميم الجرافيك.

### اختبارات الطلاب

- **اختبار 1**: اختبار أساسيات البرمجة.
- **اختبار 2**: اختبار تصميم الجرافيك.

### تفاعل مع المعلمين

- **معلم 1**: معلم برمجة.
- **معلم 2**: معلم تصميم الجرافيك.

## قواعد البيانات

### جداول قواعد البيانات

- **طلاب**: جدول للطلاب.
- **دورات**: جدول لدورات التعلم.
- **اختبارات**: جدول لاختبارات الطلاب.
- **معلمين**: جدول لماعلمين.

### أدوار المستخدمين

- **طلاب**: يمكنهم التسجيل في دورات التعلم واختبارات الطلاب.
- **معلمين**: يمكنهم إضافة دورات التعلم واختبارات الطلاب.
- **مدير**: يمكنهم إدارة المستخدمين والموارد.

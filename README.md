# منصة تعليمية لتقديم دورات إلكترونية مع اختبارات وتقويم
==============================================

## نظرة عامة وغرض المشروع
---------------

منصة تعليمية لتقديم دورات إلكترونية مع اختبارات وتقويم هي مشروع يهدف إلى توفير بيئة تعليمية إلكترونية شاملة لتقديم الدورات التعليمية مع إمكانية إجراء الاختبارات وتقويم الأداء. يهدف المشروع إلى تسهيل عملية التعلم والتدريس من خلال توفير أدوات وتقنيات حديثة ومتطورة.

## هيكل المشروع
-------------

المشروع يتكون من المكونات التالية:
* backend: يحتوي على كود الخادم ووظائف قاعدة البيانات
* frontend: يحتوي على كود الواجهة الأمامية للمستخدم
* database: يحتوي على قاعدة البيانات التي تحتوي على جميع البيانات

## تشغيل البيئة
-------------

لتشغيل البيئة، اتبع الخطوات التالية:
1. قم بتثبيت docker وdocker-compose على جهازك
2. قم بتشغيل الأمر `docker-compose up` في المجلد الرئيسي للمشروع
3. انتظر حتى يتم تشغيل جميع الخدمات
4. قم بزيارة الرابط `http://localhost:8000` في متصفحك لاستخدام المنصة

## مكونات المشروع
----------------

### الوحدات
* دورات: تحتوي على جميع الدورات التعليمية المتاحة
* اختبارات: تحتوي على جميع الاختبارات المتاحة
* تقويم: تحتوي على جميع تقييمات الأداء

### جداول
* دورات: تحتوي على معلومات حول كل دورة
* اختبارات: تحتوي على معلومات حول كل اختبار
* تقويم: تحتوي على معلومات حول كل تقييم

### أدوار
*管理员: يمكنه إدارة جميع الجوانب للمنصة
* معلم: يمكنه إدارة الدورات والاختبارات
* طالب: يمكنه المشاركة في الدورات والاختبارات
